# Jira Issue Templates

This folder contains example Jira issue payloads and guidance for common documentation workflows.

Included templates

- `tools/jira-templates/service_request_issue.json` — example payload to create a Service Request in Jira via API.

How to import

- Use the Jira REST API or create issue templates in your Jira project admin UI.

Recommended issue types

- Documentation Bug, Content Update, New Template Request, Review Request.
